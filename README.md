## API-FILA

Uma API para criar filas onde você define a fila, adiciona a pessoa na fila ou remove ela

### FUNCIONALIDADES

- Criar filas especificas
- Adicionar pessoas a uma fila
- Remover pessoas de uma fila

### COMO USAR

- Clone o projeto
```bash
git clone https://github.com/Gab-ai565/API-FILA.git
```

- Entre na pasta
```bash
cd API-FILA
```

- Instale os pré-requisitos
```bash
pip install -r requirements.txt
```

- Rode a API
```bash
python app.py
```
